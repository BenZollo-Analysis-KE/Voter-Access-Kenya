🗳️ Voter Accessibility Analysis: Kenya 2027
Project Overview
This project investigates the "Democratic Gap" in Kenya. Using GIS (Geospatial Information Systems), SQL, and Python, I am mapping how physical distance and congestion at registration centers affect youth voter participation.
The Social Impact
In high-density areas like Kibera (Nairobi) and isolated regions like the Northern Frontier, geography and infrastructure create barriers to the ballot. My goal is to identify these "Voter Deserts" to provide data-driven recommendations for the IEBC.
Technical Toolkit
QGIS: Spatial Visualization & Map Design.
PostGIS (SQL): Database management of Census 2019 records.
Python (Pandas): Automation of travel-time and turnout correlations.
Fine Arts: Cartographic aesthetics for high-impact communication.
